# chai-talwindar ☕

A fun project which is a collection of CSS utility classes, inspired by Tailwind, with fun class names.

🚀 Features
- Utility-first CSS (like Tailwind)
- No build step required
- Runs directly in the browser
- Auto-applies styles on DOM load
- Supports dynamic DOM updates (optional)

## 📦 Installation
```bash
npm install chai-css
```

## ⚡ Usage
### Import the package
```javascript
import "chai-talwindar";
```

### Add `chai` as a class
Add a class `chai` in the element you want to style.

### Use the supported classes
After including the `chai` class now you can use any supported class for styling.

## Example
```html
<div class="chai p-thodi bg-light text-center r-md">
  Hello Chai CSS ☕
</div>
```

## 🧠 How It Works
- Looks for elements with .chai
- Reads their class list
- Matches classes with predefined styles
- Applies styles using JavaScript
